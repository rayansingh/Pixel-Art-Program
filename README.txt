---- Pixel Art ----

Loading: Type the name of the 32 x 32 pixel image that you would like to load onto the grid + ".png." Then click the "Load" button

Saving: Type the name you would like to save the image as into the text field. Make sure to add ".png" to the end of it.

Clearing: Click the clear button

Erasing: Clicking the erase button will toggle the eraser on/off

Drawing/Filling: Clicking on the draw button will turn the filler off and clicking on the fill button will turn the drawer off. 

Colors: There are two color palettes that you are able to use. Click "1" or "2" to alternate.
